☕ Java Tech Quiz Game

An interactive and responsive Java Quiz Game built using HTML, CSS, Tailwind CSS, and JavaScript.

Features

- 15 Java Multiple Choice Questions
- Score Tracking
- Progress Bar
- Instant Answer Feedback
- Sound Effects
- Responsive UI
- Final Result Screen

Technologies Used

- HTML5
- CSS3
- Tailwind CSS
- JavaScript

How to Run

1. Download or clone the project
2. Open the project folder in VS Code
3. Run "index.html" in your browser

Project Purpose

This project helps users practice Java concepts through an interactive quiz interface.
